# System Requirements for TurboTax MAC software


## Operating System :
* 2021: macOS Catalina 10.15 or later

* 2020: macOS Mojave 10.14 or later

* 2019: macOS High Sierra 10.13 or later

* 2018: macOS X v.10.12 or later (64-bit required)
